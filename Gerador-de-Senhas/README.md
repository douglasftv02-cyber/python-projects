# 🔐 Gerador de Senhas

Um projeto desenvolvido em **Python** para gerar senhas aleatórias de forma rápida e segura.

## 📋 Sobre o projeto

Este programa permite ao usuário escolher o tamanho da senha e gera automaticamente uma combinação aleatória de:

- 🔤 Letras maiúsculas e minúsculas
- 🔢 Números
- 🔒 Caracteres especiais

O projeto foi desenvolvido para praticar conceitos de programação em Python, como estruturas de repetição, entrada de dados, manipulação de strings e utilização de bibliotecas.

---

## ✨ Funcionalidades

- ✔️ Geração de senhas aleatórias
- ✔️ Escolha do tamanho da senha
- ✔️ Utilização de letras, números e símbolos
- ✔️ Interface simples pelo terminal

## 🚀 Como executar

1. Clone este repositório:

```bash
git clone <URL_DO_SEU_REPOSITORIO>
```

2. Acesse a pasta do projeto:

```bash
cd Gerador-de-Senhas
```

3. Execute o programa:

```bash
python gerador_de_senhas.py
```
