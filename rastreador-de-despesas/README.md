# 💰 Expense Tracker - Python

Um sistema simples de controle financeiro pessoal desenvolvido em Python, executado via terminal. Permite registrar receitas, despesas e acompanhar o saldo em tempo real com armazenamento em arquivo JSON.

---

## 🚀 Funcionalidades

- Adicionar receitas
- Adicionar despesas
- Ver saldo atual
- Listar todas as transações
- Salvamento automático em JSON

---

## 🧠 Tecnologias utilizadas

- Python 3
- Programação orientada a objetos (POO)
- Manipulação de arquivos (JSON)
- Lógica de programação
