# 🎓 Sistema de Gestão de Alunos

Um sistema desenvolvido em **Python** para realizar o cadastro e o gerenciamento de alunos por meio do terminal.

## 📋 Sobre o projeto

Este projeto foi criado com o objetivo de praticar conceitos fundamentais da programação em Python, como listas, dicionários, estruturas de repetição, condicionais e manipulação de arquivos.

O sistema permite cadastrar, consultar, editar e excluir alunos, além de salvar os dados em um arquivo para que sejam mantidos mesmo após o encerramento do programa.

---

## ✨ Funcionalidades

- ✅ Cadastrar alunos
- ✅ Listar todos os alunos cadastrados
- ✅ Procurar aluno pelo nome
- ✅ Editar informações de um aluno
- ✅ Excluir aluno
- ✅ Salvar os dados automaticamente em um arquivo (`alunos.txt`)
- ✅ Carregar os dados ao iniciar o sistema

## 🚀 Como executar

1. Clone este repositório:

```bash
git clone <URL_DO_SEU_REPOSITORIO>
```

2. Entre na pasta do projeto:

```bash
cd Sistema-de-Gestao-de-Alunos
```

3. Execute o programa:

```bash
python sistema_alunos.py
```
